This is not a "how to do BDD" sample. This sample only aims to show you how to use bddify with NUnit. 

At the top of each code file I have provided some comments about the significance of the file. Also some methods have comments to highlight a few facts about bddify.

If you want to see a more realistic BDD sample on bddify you may install Bddify.Samples.Atm or Bddify.Samples.Bowling.
